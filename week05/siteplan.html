Ugandan Football | Site Plan

Project planning document for the Ugandan Football website.

1. Site Name

Ugandan Football

This name was chosen because the site focuses on Ugandan football news, history, and community engagement. It is direct, memorable, and clearly represents the site's content and audience.

2. Site Purpose

The purpose of this site is to provide the latest news, match day weather, team history, and a platform for fans to share suggestions to improve Ugandan football. It also serves as a hub for fixtures, team info, and membership.

3. Scenarios

  • Where can I find the latest Ugandan football news and match results?
  • How can I join a Ugandan football club or become a site member?

4. Color Schema

  • Black (#000000): Used for headings, navigation bar, and footer.
  • Yellow (#FFD700): Used for highlights, buttons, and call-to-action elements.
  • Red (#D00000): Used for accents, icons, and important notices.
  • Background (white #fff): Used for page background.

5. Typography

  • Montserrat — Used for main headings.
  • Open Sans — Used for body text and navigation.
  • Roboto — Used for subheadings and buttons.
